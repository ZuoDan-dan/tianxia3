故事背景

  万物始于混沌。混沌如卵，历一万八千年，生盘古。盘古开天辟地，清气上浮为天，浊气下沉为地。清气分为两支，西华至妙之气化为昆仑，东华至玄之气化为东海。浊气也分为两支，北极至恶之气化为幽都，南极至善之气化为生灵。大荒世界于是形成。

 昆仑山是天神西王母的居所。西王母豹首人身，管辖人间万物。西王母的侍女云华夫人为巫山神女，云华夫人游历四方，救济百姓。其追随者建立了太虚观供奉云华夫人。

 东海之外居住着天神帝俊，帝俊的属下有广成子、赤精子等十二上仙。九天玄女则是帝俊的侍女，她派出凤凰作为使者来到人间，成为后来商朝的起源。

 至善之气形成的生灵中，以人最有智慧。燧人氏发明钻木取火，教人熟食；有巢氏发明构木为巢，抵御野兽侵害。

 燧人氏有女儿华胥(xu)氏，华胥氏生下伏羲女娲两兄妹。伏羲观察天地万物的运行规律，创造了八卦。女娲取黄土和水造人，从此人不再是兽形。伏羲女娲死后飞升成仙，伏羲去了东海追随帝俊，女娲去了昆仑追随西王母。

 伏羲和女娲到少典传承了七十七代，少典两个夫人，女登生下炎帝神农氏，附宝生下黄帝轩辕氏。

 炎帝神农氏晚年体力渐衰，幽都至恶之气也危害人间，感染了许多生灵，而炎帝手下的蚩尤、刑天、雨师和风伯等将领也受到影响，变得非常残忍暴戾。炎帝心系天下，于是游走四方，遍尝百草。一路上，众多受到炎帝救助的百姓组成冰心堂，一边研习神农医术，救治百姓，一边也在寻找解除幽都之气的方法。

 炎帝遍尝百草，身中剧毒。而蚩尤、刑天、雨师和风伯已经组织了妖魔军队，意图推翻炎帝，夺取大荒统治权。炎帝于是向黄帝轩辕氏求助，共同抵抗妖魔。黄帝手下有应龙、女魃两员大将。黄帝还得到了九天玄女传授的三卷天书和风后传授的八阵图，大将应龙用八阵图训练士兵，建立了天机营；女魃用三卷天书教授弟子，建立了云麓仙居。

 炎黄和蚩尤之战终以炎黄的胜利而终结，蚩尤被杀死，其魂魄被分散封印于大荒各地。雨师和风伯带领残余的妖魔部队逃入幽都。刑天被斩首后却屹立不到，还舞动着巨斧和盾牌进行抵抗。

 经过这场大战，幽都至恶之气的影响却并未消除，不仅世间生灵遭受毒害，昆仑山上的异兽也常常受到感染。西王母于是移昆仑山于幽都之上，镇守幽都。帝俊则派出司幽设下太古铜门，并请十二上仙在门上设立天元地极锁，封印了太古铜门，幽都乃与大荒世界隔绝。位居十二上仙之首的广成子游历大荒，在巴蜀锁妖塔发现了另一个通往幽都的裂口，于是建立了弈剑听雨阁，由门下弟子负责镇守这个裂口。

 黄帝传帝位于少昊，少昊传帝位给颛(zhuan)顼(xu)，颛顼将帝位传给了帝喾(ku)，帝喾之后又传帝位给尧。


 在尧统治的时代，火神祝融与水神共工发生了一场恶战，共工力敌不过，就一头撞向不周山，不周山是昆仑山的一部分，不周山折断，整个大荒世界发生了翻天覆地的变化。不周山的碎片落在大地上，燃起无法熄灭的熊熊大火。天空倾斜，下起了倾盆大雨。而昆仑山的震动，造成了幽都的再次活动，太古铜门产生了裂痕，幽都之气泄漏出来。整个大荒陷入了前所未有的危机中。

 西王母指派火神祝融熄灭大火，于是祝融拾起散落的不周山碎片，堆放在九黎山间，成为了永不熄灭的火山。而崇拜祝融的百姓依山建立了荒火教派，供奉火神祝融。


 天空倾斜之后连降暴雨数年，引发了人间的洪水肆虐。西王母便指派女娲去补天。女娲将五色神石融化之后填补了天空的缝隙，止住了暴雨。暴雨虽止，但洪水仍在泛滥。尧便指派鲧(gun)治理大水，鲧从东海帝俊处偷来息壤，用填堵的方法治水九年而没有成功。尧处死了鲧，指派鲧的儿子禹继续治水，然后尧将帝位禅让给舜。大禹治水十三年，三过家门而不入，终于治水成功。

 大禹治水成功，舜便将帝位禅让给了德高望重的大禹。大禹继位之后励精图治，将洪水之后伤痕累累的国家治理的越来越强盛。时间匆匆过去，大禹也老了，正当子民们在想着会是谁来接替大禹成为新的帝王之时，大禹离奇的病死，而他的儿子夏启此时宣布继承自己父亲的帝位。

 夏启继位后，终于在数年之后统一大荒，建立了华夏王朝。启五十岁之时巡游到雷泽，突然离奇遇刺，朝中的几位皇子在得知父皇遇刺之后立即展开了对权力的追逐。正当他们的争夺进入白热化之时，燕丘以北的太古铜门不知被谁打开，大批的妖魔部队涌向了大荒世界。

 烽火燃遍了大荒，多个地区接连失守，被妖魔所侵占，华夏王朝岌岌可危。

 我们的世界自此而始。
